# LifeCare Hospital Website

A fully responsive Hospital Management Website built with HTML, CSS, and JavaScript. Features include:

- Home, Doctors, Appointment, and Contact pages
- Sticky navigation bar with dark/light mode toggle
- Responsive design (Flexbox & Grid)
- Doctor filtering by department (JS)
- Appointment booking with real-time validation and localStorage
- Contact form with validation
- Smooth scroll and fade-in animations (Intersection Observer)
- Professional, clean UI

## How to Use

1. Open `index.html` in your browser to start.
2. All features work client-side (no backend required).
3. Replace images in the `images/` folder with real photos as needed.

## Features
- **Dark/Light Mode:** Toggle in navbar, remembers preference.
- **Doctor Filter:** Instantly filter doctors by department.
- **Appointment Form:** Validates input, saves last appointment to localStorage, shows on home page.
- **Contact Form:** Validates and shows success message.
- **Animations:** Sections fade in as you scroll.
